# nordi_addon_carssound_fxmanifest_generator
Automatisiert die Erstellung der fxmanifest.lua für FiveM Addon Sounds. Statt jeden data_file Eintrag einzeln einzutragen scannt das PowerShell-Skript einfach deine audioconfig/ und sfx/ Ordner und generiert das komplette Manifest automatisch. Neuen Sound hinzugefügt? generate.bat doppelklicken — fertig.
